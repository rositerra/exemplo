Loja de códigos de estimação

Trabalhamos com uma linha completa de produtos, alimentos, acessórios e serviços para códigos de estimação.

Não deixamos de lado aquilo que mais importa: o bom atendimento.

Tópicos:

  • Sequência de comandos
  • Comandos Git - repositório local
  • Comandos Git - reverso

Sequência de comandos Git:

            git init
            git clone https://github.com/marciorjusto/exemplo.git
            git add .
            git commit -m "07/11/2018: Exercício 2"
            git push destino

Comandos Git - Se o repositório já existe localmente:
  1. Init
  2. Add
  3. Commit
  4. Push

Comandos Git - Se o repositório já existe localmente (reverso):
  1. Push
  2. Commit
  3. Add
  4. Init
